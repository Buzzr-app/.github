# Buzzr

Buzzr is a sports social platform and companion toolkit for the app, support pages, social automation, and DFS settlement workflows.

## Repositories

- `Buzzr`: private cross-platform product app.
- `buzzr_desktop`: public legal and support pages.
- `buzzrxautomation`: social posting automation for Buzzr.
- `dfs-engine`: pure TypeScript DFS settlement packages.
- `dfs-engine-espn`: ESPN data adapter for `dfs-engine`.
